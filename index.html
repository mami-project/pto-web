Path Transparency Observatory

An open-source, public repository for measurement of path transparency and
impairments on new and existing network protocols in the Internet.

What is Path Transparency?

Given the increasing deployment of middleboxes in the Internet, we see an ossification of the protocol stack that makes it difficult to deploy new protocols or protocol extensions. The data provided by this observatory assists to quantify impairments on path transparency when traffic is altered or treated differently on a network path based on the protocol stack in use.

This front end provides you an overview of the available measurements and some predifined visualizations of the data in the PTO. Further, the PTO offeres a more powerful restful API to enable costum analysis of the data. In order to limit the load on the PTO, only cached queries can be accessed over this frontend without an API key. If you would like to receive an API, key please contact us.

Available Measurements in the PTO

The measurement data in the PTO is mostly based on manual runs of specific studies at a certain point of interested. While repeated measurement enable to observe changes over times, e.g for ECN negotiation, it can also be see with a closer look at our results that many findings are rather stable with respect to path transparency.
